Quem inventou o sistema de controle de versão Git e por quê?
Linus Torvalds criou o Git em 2005 durante o processo de desenvolvimento do kernel do Linux. O Git era um modo de trabalhar com outras empresas e desenvolvedores com os objetivos de ter agilidade nesse processo.


Quem atualmente é o dono do Github e por quê?
A Microsoft comprou o GitHub por 7.5 bilhões de dólares em ações. O CEO da empresa ao comprar
o GitHub disse: "nos reforçamos nosso compromisso de desenvolver com liberdade e inovação"
